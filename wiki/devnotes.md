# 개발 노트

## 알려진 제한 사항

- **이미지 리사이즈 없음**: 붙여넣은 이미지 크기가 클 경우 Width/Height를 400/300으로 클램핑하지만 비율 조정은 없음
- **노트 리사이즈 없음**: 노트 카드 크기를 드래그로 조정하는 기능 미구현
- **Canvas 스크롤 없음**: 노트가 가시 영역 밖으로 나가면 접근 불가

## 설계 결정

- **이미지를 DB BLOB 대신 파일로 저장**: 스크린샷은 수백 KB~수 MB로 SQLite 성능에 영향을 줄 수 있어 파일 시스템 저장 선택
- **Canvas.Left/Top을 NoteItem.X/Y에 직접 바인딩**: ViewModel에서 X/Y를 ObservableProperty로 선언해 드래그 중 레이아웃이 즉시 반응하도록 설계
- **타이틀바만 드래그 가능**: 텍스트 노트 내용을 선택·복사할 수 있어야 하므로 콘텐츠 영역은 드래그에서 제외
